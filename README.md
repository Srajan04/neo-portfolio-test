# Portfolio Test Project

This project is a portfolio website built to test the capabilities of **Gemini 3 Pro** in generating complex, stylized web applications. It is built using **Vite** and **React**.

## Inspiration & Credits

The design and aesthetic of this project are heavily inspired by **[Marjo Ballabani](https://www.marjoballabani.me/)**.
*   **Design Credit:** Marjo Ballabani
*   **Concept:** Neo-Brutalist Portfolio

## Tech Stack

*   **AI Assistant:** Gemini 3 Pro (Preview)
*   **Framework:** React + Vite
*   **Styling:** Tailwind CSS
*   **Animations:** Lenis (Smooth Scroll), CSS Animations
*   **Icons:** Lucide React
